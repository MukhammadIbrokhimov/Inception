# Inception
Inception 42: A Docker-based project to set up a complete web stack using containers, mastering system administration, networking, and infrastructure.

## Prerequisites
- Docker
- Docker Compose
- Git
- Make

## Installation
- Clone the repository
- Run `make build`
- Run `make start`
